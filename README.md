🩺 Smart AI-Powered Health Monitoring System

📌 Overview

Smart AI-Powered Health Monitoring System is a C++ based healthcare management project designed to help users monitor and improve their daily health habits.

This project combines healthcare tracking features with AI-inspired logic to provide users with health suggestions, reminders, reports, and risk analysis in a simple console-based system.

The project was developed as a hackathon and portfolio project with the aim of combining healthcare and technology.
